# Jhonny Data Integrity Portfolio

Portafolio profesional de Jhonny Daniel - Ingeniero de Sistemas.
Desarrollado con arquitectura hexagonal: React + Node.js + MySQL.

## Tecnologias
- Frontend: React + Vite
- Backend: Node.js + Express
- Base de datos: MySQL (Navicat 17)
- Arquitectura: Hexagonal (Ports and Adapters)

## Estructura
- /frontend  -> Aplicacion React
- /backend   -> API REST Node.js
- /database  -> Scripts SQL
